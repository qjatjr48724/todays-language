## 2. 제품 개요

- **한 줄:** 초보자도 부담 없이, **매일 조금씩** 다양한 언어를 학습할 수 있게 돕는 **AI 기반 데일리 학습 앱**.
- **핵심 가치:** 적은 양 + 매일 반복 + 쉬운 접근성. 유료 강의/앱 부담을 줄이고 **가능한 무료** 제공을 지향 (상황에 따라 유료 전환 가능성은 열어 둠).
- **초기 타깃:** 한국인. **추후** 전 세계 사용자 — **모국어(로컬 언어)** 와 **학습 언어**를 각각 설정.
- **초기 학습 언어:** **한국어 / 일본어** 고정. 한국인에게 한국어는 모국어이므로 **단어 의미 중심**으로 학습.
- **추후 (한국어 학습 모드):** 단어·문장·속담(+ 의미) 등 확장.

## 3. 기술 스택 (대화에서 확정)

| 영역 | 선택 | 비고 |
|------|------|------|
| **모바일** | **Flutter** | 두 스토어(Android/iOS) 목표, 1인 개발에 적합 |
| **백엔드** | **Firebase** | Auth, Firestore 등. DB를 직접 서버에 깔아 관리하는 단계는 당분간 피함 |
| **AI** | **Firebase Cloud Functions**(또는 동급 서버)에서만 외부 API 호출 | **보안:** API 키는 앱에 넣지 않음 |

### 개발 환경 메모

- 주로 **Windows**, **Mac**에서도 작업 가능.
- **iOS** 빌드·스토어 제출은 **Mac + Xcode** 필요.
- 폴더 이름 변경은 가능; 변경 후 IDE에서 새 경로로 폴더 다시 열 것.

## 7. 메뉴·기능 (초기 홈 기준)

### 홈 상단

- **좌상단:** 내 정보
- **우상단:** 언어 변경 (초기에는 로컬=한국어, 학습=일본어 **고정**)

### 내 정보에 들어갈 내용 (기획)

- 이름, 국가(로컬 언어 기준), 최초 가입일(최초 로그인일), 로그인 방식(이메일은 앱 로고 등, 소셜은 해당 아이콘)
- **전체 데이터 백업** (기기 변경·클라우드)
- **좌하단:** 리뷰 작성 (Android → Play, iOS → App Store 리뷰 페이지)

### 홈 중간

1. **오늘의 단어** — 하루 **50개**; 매일 새 단어 + 이전 학습 단어 **일정 비율** 복습
2. **오늘의 문장** — 하루 **10개**; 문장 길이 상한(장문 지양); 해석 시 **품사별 색** 구분
3. **오늘의 속담** — **공부 언어 선택 가능 + 공부 언어가 한국어일 때만** 표시; **그 외 비활성·숨김** → **초기에는 숨김 및 비활성**
4. **단어 퀴즈** — 상단에 로컬 언어로 뜻, **4지선다**, 정답 포함; **발음 듣기** 추가

**공통 (1~3):** 단어 + 로컬 언어 발음 표기 + 해석 + 발음 듣기  
**4번:** 발음 듣기

### 홈 하단

- **오늘의 진행률** — 사용자가 정리한 규칙: **진행률 구간별 색** (예: 낮음→높음 방향으로 빨/노/초 등 의도에 맞게 조정함; 세부는 Notion 기준 최신본 확인)

### 개발 단계 (메뉴 외)

- **중반:** 로컬/학습 언어 선택, 언어별로 가능한 메뉴·화면 변화
- **후반:** 소셜 로그인 확장 등
